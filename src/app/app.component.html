COMPONENT FAMILIES COMMUNICATE ~ASYNCHROSOUSLY~

    What's Going On

  • Set the MOOD of each family member using the SLIDER.
  • Families rely on each other, so Average Family Happiness effects the individual happiness of each family member (according to a weighted score)

    • How It Works

  • Using the Observer pattern, components at any point in the hierarchy subscribe to the data changes of their family members. Data is communicated immediately without passing props or using the Store pattern

    • Use Cases

  • Cross-form validation, complex attribute pickers, scoring systems......
